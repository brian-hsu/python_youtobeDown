#### 這個應用是**python製作的CLI**用來儲存或者轉檔 Youtobe 影片。

**動機:**因為線上轉檔網站有時候會不穩。

有build一個**Windows執行檔**(pytube_mian.exe)可以供大家試試。

**這個小程式執行後只有幾個選項:**

1.複製 youtube 網址。

2.如需要轉 mp3 檔, 打上 "y"。

3.如需要轉 mpeg 檔, 打上 "y"。

4.如需要保留原 mp4 檔, 打上 "y"。

![](https://images2.imgbox.com/f0/49/Ivs5dqdu_o.png)

如果有轉 mpeg 檔，原檔(MP4)會刪除，除非選擇保留mp4 檔。

有選擇轉 mp3 檔預設是保留原檔(MP4)。
